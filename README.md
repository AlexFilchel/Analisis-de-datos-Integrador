🔹 Eliminación de duplicados

Se eliminaron registros duplicados para evitar distorsiones en el análisis estadístico.

🔹 Corrección de tipos

Se transformaron valores no numéricos a formato adecuado, permitiendo su análisis cuantitativo.

🔹 Tratamiento de valores nulos

Se eliminaron registros sin variable objetivo y se imputaron valores faltantes utilizando la mediana, por su robustez frente a valores extremos.

🔹 Filtrado de outliers

Se aplicaron reglas de negocio para eliminar valores fuera de rangos realistas (ej. horas de estudio o sueño).

🔹 Ordenamiento temporal

Se ordenaron los datos por estudiante y semana para garantizar la coherencia en el análisis longitudinal.
